# R Project Archive
### A Three-Layer Time Capsule of Contemporary Parsi Discourse

---

## Overview

This repository contains the **R Project Archive**—a structured, multi-layered collection of contemporary Parsi/Zoroastrian online discourse.

The project captures real conversations and preserves them across multiple linguistic and interpretive layers, creating a **time capsule of how the community expresses itself in the present moment**.

> This is not a preservation or revival effort.  
> It is a structured archival and interpretive experiment.

---

## Core Concept

Each entry in this archive represents a single conversation, captured and transformed across multiple layers:

1. **Screenshot (Primary Layer)**  
   The original conversation as it appeared online.

2. **Transcription (Source Text)**  
   Cleaned, readable version of the original content.

3. **Esperanto Layer (Neutral Transformation)**  
   A structurally simplified rendering for clarity and normalization.

4. **Constructed Language Layer (Interpretive Transformation)**  
   A stylized, expressive reinterpretation using a custom-designed conlang.

5. **Additional Translations (Optional)**  
   Hindi, Gujarati, or other languages where available.

Together, these layers allow comparison between:
- raw expression  
- normalized meaning  
- stylized interpretation

## Methodology

The workflow is intentionally simple and modular:

```

Screenshot → Transcription → Esperanto → Conlang → Optional translations

```

This enables distributed contribution without requiring full system knowledge.

## Contributions

This project is designed for lightweight, role-based contributions.

Typical roles include:

- **Content Collection** → capturing public discussions (screenshots)  
- **Transcription** → converting screenshots into text  
- **Translation** → Hindi, Gujarati, etc.  
- **Curation** → structuring entries and maintaining consistency  

See `CONTRIBUTING.md` for details.
